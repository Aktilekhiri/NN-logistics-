# NN Logistics — Корпоративный сайт

Современный одностраничный сайт для логистической компании **NN Logistics** (Бишкек, Кыргызстан), созданный на основе данных из 2ГИС.

---

## О проекте

Сайт разработан как демонстрационный прототип для компании, у которой отсутствует онлайн-присутствие. Все контактные данные, адрес и рейтинг взяты из реального профиля на 2ГИС.

**Рейтинг компании на 2ГИС:** ⭐ 5.0 (12 оценок)  
**Адрес:** ул. Манаса 180, 3 этаж, Бишкек  
**Телефон:** +996 995 666 896

---

## Технологии

- Чистый HTML5 / CSS3 / Vanilla JavaScript
- Никаких фреймворков и зависимостей — один файл
- Google Fonts: `Bebas Neue` (заголовки) + `Outfit` (текст)
- CSS-анимации и IntersectionObserver API

---

## Структура страницы

| Секция | Описание |
|---|---|
| **Nav** | Фиксированная навигация, меняет фон при скролле, мобильное меню |
| **Hero** | Главный экран с анимированными заголовком, описанием и статистикой |
| **Ticker** | Бегущая строка с перечнем услуг |
| **Services** | 6 карточек услуг с hover-эффектами |
| **Why Us** | Преимущества компании + визуальный блок |
| **Reviews** | Отзывы клиентов на основе данных 2ГИС |
| **Contact** | Контакты + форма заявки |
| **Footer** | Ссылки и копирайт |

---

## Особенности дизайна

- **Тема:** Тёмная, индустриальная — чёрный фон с золотым акцентом `#f5a623`
- **Кастомный курсор** с плавным следованием
- **Scroll-reveal анимации** — элементы появляются при прокрутке
- **Stagger-эффект** — карточки появляются поочерёдно
- **Адаптивная вёрстка** — корректно отображается на мобильных устройствах

## Запуск

Просто откройте файл в браузере:

```bash
open nn-logistics.html
```

Никаких сборщиков, серверов и установки зависимостей не требуется.

---

## Как передать клиенту

1. Отправьте файл `nn-logistics.html` напрямую
2. Для публикации в интернете — загрузите на любой хостинг (Netlify, GitHub Pages и т.д.)
3. Для полноценного запуска рекомендуется подключить реальную карту через 2ГИС API или Google Maps
